# Click Runner

## Descripción

Click Runner es una experiencia interactiva que transforma la velocidad de los clicks de un usuario en una animación visual personalizada.

A diferencia de los tradicionales CPS (Clicks Per Second) Tests, Click Runner no solo muestra una puntuación numérica, sino que convierte el ritmo de los clicks en el movimiento de un personaje animado, permitiendo al usuario visualizar su rendimiento de una manera única y entretenida.

---

## Objetivo

Crear una herramienta divertida que combine:

* Medición de velocidad de clicks.
* Visualización en tiempo real.
* Reproducción personalizada del rendimiento.
* Resultados fáciles de compartir.

---

## Cómo funciona

1. El usuario selecciona una duración para la prueba.
2. Comienza el contador.
3. Durante el tiempo establecido, el usuario realiza la mayor cantidad de clicks posible.
4. El sistema registra:

   * Cantidad total de clicks.
   * CPS (Clicks Per Second).
   * Intervalos entre cada click.
5. Al finalizar:

   * Se muestra la puntuación.
   * Se asigna una categoría o estilo.
   * Se reproduce una animación basada en el ritmo registrado.

## Filosofía del proyecto

La mayoría de los tests de clicks terminan mostrando únicamente un número.

Click Runner busca transformar ese número en una experiencia visual, permitiendo que cada prueba genere una representación única del ritmo y comportamiento del usuario.

Cada sesión produce una animación diferente porque cada persona hace click de una manera diferente.
